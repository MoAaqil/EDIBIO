1. STRUCTURAL CHANGES (RESTACKING)
------------------------------------------------------------------------
- Root directory cleaned: Deleted unused log and compile output files.
- Standalone maintenance scripts moved to backend/scripts/ to reduce clutter.
- The project has been physically split into two main sections:
  * frontend/ : Houses the Next.js runtime, UI layouts, components, public assets, Capacitor native wrappers, and Electron configuration.
  * backend/  : Houses the database models and connection helpers (lib/) and utility scripts (scripts/).
- Integrated "copy-backend.js": Since Next.js requires TypeScript imports to be local, this pre-build script copies Mongoose models from the backend directory to the frontend library directory prior to compilation.
- Refactored package.json scripts to automate copying.
